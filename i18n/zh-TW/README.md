## Twitterᴾˡᵘˢ
增強Twitter使用體驗。讀取原始畫質的圖片，移除廣告與垃圾推文。

#### 主要功能
* 讀取原始畫質的圖片
  * 從右鍵選單複製圖片網址或儲存都使用原始畫質。
* ~~允許從推文下載影片 (必須在該影片的推文中)~~
  * 由於影片下載服務不穩定，已經移除此項功能。
* 移除包含特定Hashtag或Hashtag超過最大限制的推文 (最大限制預設20)
* 移除官方廣告的推文 (測試中)
